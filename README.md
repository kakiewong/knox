# Knox
A simple budget tracker
<br>
<br>
This project utilizes CRUD operations to build a user-friendly expense tracking web app. Users can add transactions and see in real-time changes to their budget. They can also delete entries from their list of transactions. The breakdown view shows users where they are spending their money with an easy to read graph.
